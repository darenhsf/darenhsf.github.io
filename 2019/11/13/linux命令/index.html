rm 命令

-i 删除前逐一询问确认：

-r 选项，删除目录

rmdir 命令

从一个目录中删除一个或多个子目录项，删除某目录时也必须具有对其父目录的写权限。

mv 命令

移动文件或修改文件名，根据第二参数类型（如目录，则移动文件；如为文件则重命令该文件）

当第二个参数为目录时，第一个参数可以是多个以空格分隔的文件或目录

cp 命令

将源文件复制至目标文件，或将多个源文件复制至目标目录。

cat 命令

  • -b 对非空输出行号
  • -n 输出所有行号

搜索

在 linux 要查找某个文件，但不知道放在哪里了，可以使用下面的一些命令来搜索

  • which 是在 PATH 就是指定的路径中，搜索某个系统命令的位置，并返回第一个搜索结果。
  • whereis 命令只能用于程序名的搜索，而且只搜索二进制文件（参数-b）、man说明文件（参数-m）和源代码文件（参数-s）
  • locate 通过搜寻系统内建文档数据库达到快速找到档案

find

实际搜寻硬盘查询文件名称。

find pathname -options [-print -exec -ok …]

chmod 命令

改变 linux 系统文件或目录的访问权限

每一文件或目录的访问权限都有三组，每组用三位表示，分别为文件属主的读、写和执行权限；与属主同组的用户的读、写和执行权限；系统中其他用户的读、写和执行权限。可使用 ls -l test.txt 查找。

tar 命令

tar 本身不具有压缩功能，只具有打包功能

chown 命令

chown 将指定文件的拥有者改为指定的用户或组

grep 命令

强大的文本搜索命令，grep(Global Regular Expression Print) 全局正则表达式搜索。

grep 的工作方式是这样的，它在一个或多个文件中搜索字符串模板。如果模板包括空格，则必须被引用，模板后的所有字符串被看作文件名。搜索的结果被送到标准输出，不影响原文件内容。
