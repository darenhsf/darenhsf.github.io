    javascript

    JavaScript是基于浏览器、基于面向对象、事件驱动式的网页脚本语言。
    1.基于浏览器:

    • Javascript他是被浏览器来进行解析的.HTML、CSS、 Javascript都是被浏览器解
      析的.

    2.基于面向对象:

    • 在Javascript中- -切皆对象,对象由属性和方法组成!

    3.事件驱动式:

    • 用户的某些行为,比如鼠标经过、鼠标离开、鼠标点击等这些行为他会被
      Javascript所捕获,Javascript可以对这些行为赋予一些功能代码.

    Javascript书写格式

    1. 嵌入式:
    • 通过标签将Javascript代码嵌入到HTML代码中
    1. 外链式:
    • JS代码它可以单独的保存为一个以.js为扩展名的文件然后通过HTML中的的src属性将其引入到当前的HTML文件中!
    1. 行内式:
    • 将JS代码书写在HTML标签的事件属性中
      HTML标签中有一些事件属性带有on前缀比如鼠标经过、鼠标离开、鼠标单击等
      <标签名事件属性=“JS代码”>

    Javascript语法规则

    1. JavaScript区 分大小写
    2. JavaScript脚本程序须嵌入在HTML文件中
    3. JavaScript脚本程序中不能包含HTML标记代码
    4. 每行写一条脚本语句
    5. 每一条语句末尾可以加分号-定是英文下的分号(;),最好加分号
    6. JavaScript脚本程序可以独立保存为一-个外部文件这个文件是不能自己运行的它
      必须要依赖于HTML文件

    2.window.alert(“要输出的内容”):

    host

    本地解析文件

    Hosts文件主要作用是定义IP地址和主机名的映射关系，是一个映射IP地址和主机名的规定。

    是一个没有扩展名的系统文件，它的主要作用是能加快域名解析，还可以屏蔽网站等

    在访问dns之前，先访问这个文件。

    c:\windows\system32\drivers\etc\hosts
