proxmark3

PM3常用命令

连接PM3 proxmark3

系统测试

  • 天线调谐测试 hw tune
  • 显示系统版本hw version

卡读取

  • 通用低频卡读取 lf search
  • em410系低频卡读取 lf em4x em410xwatch
  • 高频卡类型读取 hf 14a reader
  • t5577卡检测 lf t55xx detect
  • t5577卡读取 lf t55xx dump

密钥获取

  • 密钥查询hf mf chk |<*card memory> [t|d] [] []
  • 检测密码 hf mf chk 0 A ffffffffffff
  • 检测默认密码 hf mf chk * ? [t]
  • PRNG 攻击 hf mf mifare
  • 开启离线嗅探 hf 14a snoop
  • 查看离线嗅探数据 hf list 14a
  • Nested 攻击 hf mf nested 1 0 A ffffffffffff [d]

数据导出

  • dump卡片数据 hf mf dump
  • bin转换为eml文件 script run dumptoemul.lua

数据写入

  • 指定块数据写入 hf mf wrbl
  • UID卡整卡克隆hf mf cload filename
  • T5577卡写入id卡uid号 lf em4x em410xwrite 1 64
  • T5577卡扇区写入数据 lf t55xx wr [password]
