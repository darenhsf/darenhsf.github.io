一些命令添加sudo后找不到

发表于 2020-08-02 更新于 2021-10-14 分类于 linux 阅读次数：

sudo 命令执行找不到命令

如

sudo scapy

原因

sudo是将用户切换为root,但是由于root并不是scapy文件的拥有者，找不到scapy的路径

解决方法

以scapy为例

1.配置sudo的配置文件中的secure_path

2.或者直接

3.或者为 scapy路径 加一个软链接到sudo对应路径下
